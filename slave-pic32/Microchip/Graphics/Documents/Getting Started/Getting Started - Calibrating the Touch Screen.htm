All the demos that feature a resistive touch screen can be calibrated. Calibration is needed since resistance characteristics of the touch screen may vary from one unit to another due to manufacturing process and environmental factors over time. Environmental factors such as humidity and temperature may alter the resistance of the materials which affects the measurement of the position of the touch.

 

The calibration sequence is the same for all hardware configurations. The only difference is the actual hardware component to use to enable the calibration.

 

Configuration 1: PIC24FJ256DA210 Development Board. 1

Configuration 2: Explorer 16. 1

Configuration 3: Graphics LCD Controller PICtail™ Plus SSD1926 Board with PIC32 Starter Kits  2

Configuration 4: Multimedia Expansion Board. 2

 

  Configuration 1: PIC24FJ256DA210 Development Board

To calibrate:

1)       Hold push button S1

2)       While holding down S1 press and release the MCLR  (S4) button. This will execute the screen calibration.

 

3)       Follow the on screen instructions to calibrate.

  Configuration 2: Explorer 16

To calibrate:

1)       Hold push button S3

2)       While holding down S3 press and release the MCLR (S1) button. This will execute the screen calibration.

 

 

3)       Follow the on screen instructions to calibrate.

 

 

Configuration 3: Graphics LCD Controller PICtail™ Plus SSD1926 Board with PIC32 Starter Kits

Configuration 4: Multimedia Expansion Board

 

When using the PIC32 Starter Kits with the Graphics LCD Controller PICtail™ Plus SSD1926 Board or Multimedia Expansion Board, calibration is a little bit tricky. Since there is no MCLR button on the Starter Kits the calibration should be performed with care.

 

1)       Hold push button SW1 on the PIC32 Starter Kit.

2)       While holding down the SW1 carefully power up the Starter Kit using the USB connector or the 9V power supply ((J5) for the Graphics LCD Controller PICtail™ Plus SSD1926 Board, (J1) for the Multimedia Expansion Board).

 

3)       Follow the on screen instructions to calibrate.
